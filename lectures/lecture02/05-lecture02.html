14.1 Basic Plot Setup

import matplotlib.pyplot as plt
import numpy as np

# Create a new figure
plt.figure(figsize=(width, height))

# Create a plot
plt.plot(x, y, 'bo-')  # Blue line with circle markers

# Set labels and title
plt.xlabel('X-axis label')
plt.ylabel('Y-axis label')
plt.title('Plot Title')

# Add a legend
plt.legend(['Label'], loc='best')

# Show the plot
plt.show()

14.2 Common Plot Types

  • Line plot: plt.plot(x, y)
  • Scatter plot: plt.scatter(x, y)
  • Bar plot: plt.bar(x, height)
  • Histogram: plt.hist(data, bins=10)
  • Box plot: plt.boxplot(data)
  • Errorbar plot: plt.errorbar(x, y, yerr=error)

14.3 Customization

  • Set axis limits: plt.xlim(xmin, xmax), plt.ylim(ymin, ymax)
  • Set axis scales: plt.xscale('log'), plt.yscale('log')
  • Set tick marks: plt.xticks(ticks, labels), plt.yticks(ticks, labels)
  • Add a grid: plt.grid(True)
  • Change line style: plt.plot(x, y, linestyle='--', color='r', linewidth=2)
  • Change marker style: plt.plot(x, y, marker='o', markersize=5)

14.4 Multiple Plots

  • Subplots: fig, (ax1, ax2) = plt.subplots(1, 2, figsize=(10,5))
  • Plot on specific axes: ax1.plot(x, y)

14.5 Saving Plots

plt.savefig('filename.png', dpi=300, bbox_inches='tight')

14.6 Advanced Plots

  • Contour plot: plt.contour(X, Y, Z)

  • 3D plot:

    from mpl_toolkits.mplot3d import Axes3D
fig = plt.figure()
ax = fig.add_subplot(111, projection='3d')
ax.plot_surface(X, Y, Z)

14.7 Useful Settings

  • Adjust layout: plt.tight_layout()
  • Set style: plt.style.use('ggplot')
  • Colormap: plt.imshow(data, cmap='viridis')
